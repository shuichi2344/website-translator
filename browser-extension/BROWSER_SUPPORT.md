# Browser Support Summary

## ✅ Fully Supported Browsers

### Chrome, Brave, Edge, Opera (Chromium-based)
- Install the extension directly
- Automatic government website detection
- One-click summarization

### Safari, Firefox, and ALL other browsers
- Use the universal bookmarklet
- Same features as the extension
- No installation required
- Works on mobile too!

## Quick Setup

### For Chrome/Brave/Edge/Opera:
1. Go to your browser's extensions page
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `browser-extension` folder

### For Safari/Firefox/Any Browser:
1. Create a new bookmark
2. Name it "📄 Summarize Page"
3. Paste the bookmarklet code from CROSS_BROWSER.md
4. Click it on any government website

## Features (All Browsers)
- ✅ Auto-detect ASEAN government websites
- ✅ Crawl 3 sublinks automatically
- ✅ Simple 5th-grade language summaries
- ✅ 12 ASEAN languages supported
- ✅ Copy & Read Aloud functions
