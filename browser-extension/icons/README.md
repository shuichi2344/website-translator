## Icon Design

- Background: Purple gradient (#667eea)
- Icon: Simple document symbol with lines
- Style: Flat, modern, minimal
- Colors: Match the app theme

## Customizing Icons

If you want to create custom icons:

### Option 1: Use an online tool
1. Go to https://www.favicon-generator.org/
2. Upload a logo or create one
3. Download the generated icons
4. Rename them to match the required sizes

### Option 2: Use design software
- Use Figma, Canva, or Photoshop
- Create a simple icon with:
  - Document/page symbol 📄
  - Bridge or connection symbol 🌉
  - Government building 🏛️
- Export in 16x16, 48x48, and 128x128 sizes

### Option 3: Use emoji as placeholder
For quick testing, you can use emoji-to-image converters:
1. Go to https://emoji-to-png.com/
2. Convert 📄 emoji to PNG
3. Resize to required dimensions

## Recommended Design

- Background: Gradient (purple/blue like the app)
- Icon: Simple document or bridge symbol
- Style: Flat, modern, minimal
- Colors: Match the app theme (#667eea to #764ba2)
